## 二、完整题库（48 道）

### 题目数据结构说明

```typescript
interface Question {
  id: number;              // 题号 1-48
  text: string;            // 题目文本
  dimension: 'E' | 'I' | 'S' | 'N' | 'T' | 'F' | 'J' | 'P';  // 所属半维度
  direction: 1 | -1;       // 1=正向(同意→该半维度加分), -1=反向(同意→对面半维度加分)
  usedInQuick?: boolean;   // 是否入选快速测试12题
}
```

**计分速查**：
- `direction: 1` → 用户选"同意"(+1/+2/+3) → 该 dimension 得正分
- `direction: -1` → 用户选"同意"(+1/+2/+3) → **对面的** dimension 得正分（即本 dimension 得负分）

## 四、人格类型详细档案

### 4.1 快速测试 4 种 — 详细描述

#### 🦋 BUTTERFLY 社蝶（E + S）

> 你是一只飞舞在南方的社交蝴蝶

你随性而不随便。你看起来很佛系（"都可以呀""没关系啦"），但其实你有很坚定的审美标准和价值观。触碰到底线的事情你会温和但坚定地拒绝。你的温柔不是软弱，是一种选择。

**南方名场面**：艺术节海报设计者、班级板报永远是你画的、被公认"品味最好"的人

#### 1️⃣5️⃣ STRATEGIST 军师（INTJ）

> **"按我的计划，第三步反转"**

你是南方最高冷的学霸，一个行走的战略中枢。当别人在为一道数学题苦恼的时候，你已经在想这道题可以延伸出哪些变式、对应哪些考点、高考可能的命题方向是什么。

你的世界由系统和逻辑构成。你喜欢把所有事情都纳入一个框架：学习方法是一个系统、时间管理是一个系统、人际交往也是一个系统（虽然这个系统你运行得不太熟练）。你不太擅长（也不太感兴趣）闲聊和社交，但如果讨论的是一个有价值的话题，你可以滔滔不绝讲两个小时。

你的独立性强到让老师有时候都不知道该怎么管你——你不太听话，但你的成绩和产出又让人挑不出毛病。你按照自己的节奏和方式前进，效果往往比按部就班的人还好。这让那些"乖学生"很困惑也很羡慕。

**南方名场面**：年级排名前列但从不刷题的那种人、解题方法总是和老师教的不一样但更快、被形容为"高冷但有实力"

#### 1️⃣6️⃣ PROPHET 先知（INFJ）

> **"你今天看起来不开心，发生什么了？"**

你是南方最稀有的存在——据统计只有约1-2%的人属于这种类型。你有一种近乎超自然的洞察力，能够感知到周围人的情绪和需求，甚至在对方自己都没有意识到之前。

你看起来安静温和，但你的内心世界极其丰富且复杂。你经常思考一些深刻的问题：人生的意义、公平与正义、人与人的连接。这些思考让你显得比同龄人"成熟"（也有人说是"老灵魂"）。

你在南方扮演的角色往往是"无声的领导者"——不是通过命令和权威，而是通过理解和共情来影响身边的人。你不需要站出来说话，你的存在本身就是一种力量。但这也意味着你承担了很多看不见的情绪负担。请记得：先知也需要休息。

**南方名场面**：被很多人信任并倾诉心事、说话不多但每次开口都让人醍醐灌顶、频道里写出深度长文引发大量共鸣

### 4.3 隐藏款 3 种 — 详细档案

#### 🐱 H1: MEOW 猫学长（隐藏款）

你独立、自由、神秘。你想出现的时候就出现（比如晚自习后突然出现在操场一角），想消失的时候就消失（整个假期没有人知道你去哪了）。你对人群保持着一种微妙的距离——不抗拒但也不主动靠近，像是在观察人类这种有趣生物的外星访客。

你有很强的同理心和感受力，但你表达关心的方式很独特：不是嘘寒问暖，而是在对方最需要的时候恰好出现，然后什么都不说，只是待在那里。

你在南方是一个传说。不是每个人见过你，但见过你的人都不会忘记。**恭喜你解锁了南方最稀有的人格——猫学长本人。**

*解锁台词："喵。"（翻译：你懂的就懂。）*

#### 🍱 H2: CANTIN 食堂战神（隐藏款）

**触发条件**：高 S（感觉）+ 高 J（判断）+ 隐藏题关于食堂的极端回答命中

你不仅是吃货，更是食堂生态系统的观察者和记录者。吃出异物？你第一时间拍照发帖维权。菜品涨价？你制作了详细的年度价格走势图。早餐种类减少？你写了一篇《南方早餐消亡史》引发全频道热议。

你对食堂的爱是深沉的、理性的、数据驱动的。有人说你"太关注吃的了"，但你知道——在南方，食堂就是第二个家。**恭喜你解锁了食堂战神称号！**

*解锁台词："三楼梅菜扣肉，十块钱，白月光。不接受反驳。"*

你是南方的混沌实体，一个美丽的错误。**恭喜——你稀有到连算法都放弃了给你贴标签。**
