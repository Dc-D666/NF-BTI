## 一、v1 → v2 修改说明

### 1.1 解决的相似度问题

| v1 问题 | v2 解决方案 |
|---------|-----------|
| Q17/Q18 同一句话两种解读 | 拆为两个完全不同的场景 |
| Q19/Q20 同一只猫两种描述 | 拆为完全不同的 S/N 场景 |
| Q27/Q28 同一室友台灯场景 | 分别换为"手机被查"和"举报同学"两个独立场景 |
| Q31/Q32 同一外卖被偷场景 | 分别换为"双休被取消的分析"和"朋友受伤的反应" |
| T/F 维度4组都是"同一事件不同反应" | 重构为 6 个完全独立的场景 |
| Q35 刘春虎老师具体名字 | 改为年级主任的具体管理措施 |

### 1.2 v2 设计原则

1. **同维度内每道题的场景完全不重复**（v1 最大问题）
2. **每个维度覆盖 6 个不同的生活领域**（学习/社交/饮食/生活/规则/情感）
3. **融入新增素材中的独特发现**（手机藏匿、堵桥狗、研学、阳光体育、crush 等）
4. **正反题（direction=±1）在语义上真正互补而非同场景翻转**
